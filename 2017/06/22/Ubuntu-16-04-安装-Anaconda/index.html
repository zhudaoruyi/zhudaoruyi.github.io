Ubuntu 16.04 安装 Anaconda

发表于 2017-06-22

参考 docs.continuum.io/anaconda/install-linux.html

1、下载适合你电脑的版本。

按照不同的版本分类，docs.continuum.io/anaconda/hashes/ 含有md5值 和 SHA-256值
所有的历史版本列表，repo.continuum.io/archive/index.html 含有md5值

2、验证 MD5 或 SHA-256。

运行

或

例如下载 Anaconda3-4.2.0-Linux-x86_64.sh 到 Downloads 文件夹中后

3、运行sh文件，进行安装。

按enter，输入yes一直默认安装完成。
4、退出当前的shell终端。
再启动终端时就可以输入 conda list查看 python 包列表。
